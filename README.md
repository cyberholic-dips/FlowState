# Habit Tracker - React Native App

A minimal, personal mobile app for tracking daily habits, writing quick notes, and measuring focus time.

## Getting Started

### Prerequisites
- Node.js installed
- Expo CLI
- Expo Go app on your mobile device (for testing)

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Scan the QR code with Expo Go (Android) or Camera app (iOS)

## Features

- **TODAY**: Track your daily habits
- **STATS**: View your progress and statistics
- **EXPLORE**: Discover new habits to build
- **SETTINGS**: Customize your experience

## Tech Stack

- React Native
- Expo
- React Navigation (Bottom Tabs)
- Expo Vector Icons
